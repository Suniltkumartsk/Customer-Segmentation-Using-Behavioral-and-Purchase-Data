# Customer Shopping Behavior Analysis

This project analyzes customer shopping behavior using SQL queries and Python data analysis in a Jupyter Notebook.

## Project Structure

- **Customer_Shopping_Behavior_Analysis.ipynb**: Main Jupyter Notebook for data analysis, visualization, and insights.
- **customer_behavior_sql_queries.sql**: SQL queries used for data extraction and transformation.

## How to Use

1. Open the Jupyter Notebook (`Customer_Shopping_Behavior_Analysis.ipynb`) in VS Code or Jupyter Lab.
2. Run the notebook cells to perform data analysis and view visualizations.
3. Use the SQL queries in `customer_behavior_sql_queries.sql` to extract or manipulate data as needed.

## Requirements

- Python 3.x
- Jupyter Notebook or Jupyter Lab
- Common Python data analysis libraries (e.g., pandas, matplotlib, seaborn)
- SQL database (if running queries)

## Overview

The analysis covers:
- Data cleaning and preprocessing
- Exploratory data analysis (EDA)
- Visualization of customer shopping patterns
- Insights and recommendations based on data trends
